TensorFlow: Hello World!

TensorFlow is a Python library for fast numerical computing created and released by Google. It is an open-source library primarily for deep learning applications. It also supports traditional machine learning. TensorFlow was designed for use both in research and development and in production systems

Getting Tensorflow

A comprehensive guide to download and install tensorflow is available here. However, you can run tensorflow in Google Colaboratory. Colab proves jupyter like environment where you can interactively edit and execute notebooks on Google's cloud servers. These servers run via virtual machine with access to CPU, GPU and even TPUs. If you are already familiar with iPython jupyter notebooks, the rest should be easy.
