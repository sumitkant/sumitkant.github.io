# Coolers Modern Skill

This skill provides a curated collection of 48 professional font and color themes, each with a carefully selected color palette and font pairing. Once a theme is chosen, it can be applied to any artifact.

## Purpose

To apply consistent, professional styling to presentation slide decks, documents, reports, and web pages, use this skill. Each theme includes:
- A cohesive color palette with hex codes
- A set of role assignments (background, body text, primary accent, secondary accent)
- Complementary font pairings for headers and body text
- A distinct visual identity suitable for different contexts and audiences

## Usage Instructions

To apply styling to a slide deck or other artifact:

1. **Show the theme showcase**: Display the `theme_showcase.pdf` file to allow users to see all available themes visually. Do not make any modifications to it; simply show the file for viewing.
2. **Ask for their choice**: Ask which theme to apply to the artifact.
3. **Wait for selection**: Get explicit confirmation about the chosen theme.
4. **Apply the theme**: Once a theme has been chosen, apply the selected theme's colors and fonts to the artifact.

## Typography

Every theme in this collection shares the same font pairing for a consistent, modern feel:
- **Headers:** Plus Jakarta Sans
- **Body Text:** Inter

## Theme Roles

Each theme defines four roles that map onto an artifact:
- **Background** - chosen page and canvas background
- **Body Text** - primary readable text color
- **Primary Accent** - buttons, links, and key visual emphasis
- **Secondary Accent** - highlights, badges, charts, and supporting details

## Application Process

After a preferred theme is selected:
1. Use the four role colors listed for that theme (background, body text, primary accent, secondary accent).
2. Set headers in Plus Jakarta Sans and body copy in Inter.
3. Apply the specified colors and fonts consistently throughout the artifact.
4. Ensure proper contrast and readability.
5. Maintain the theme's visual identity across all slides or sections.

## Create your Own Theme

To handle cases where none of the existing themes work for an artifact, create a custom theme. Based on the inputs provided, generate a new theme similar to the ones above. Give the theme a name describing what the font/color combination represents. Use any basic description provided to choose appropriate colors and fonts, and define the same four roles (background, body text, primary accent, secondary accent) plus a header/body font pairing. After generating the theme, show it for review and verification. Following that, apply the theme as described above.
