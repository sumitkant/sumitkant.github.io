# Silent Waters

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Pale Teal**: `#E0FBFC` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Pale Teal 2**: `#C2DFE3` - Soft backgrounds, highlight panels, and secondary content blocks
- **Cool Cyan**: `#9DB4C0` - Primary interface accents, links, charts, and trust-building UI elements
- **Balanced Cyan**: `#5C6B73` - Primary interface accents, links, charts, and trust-building UI elements
- **Deep Cyan**: `#253237` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
