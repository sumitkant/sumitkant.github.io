# Classic Red Palette

A broad-spectrum color system that feels modern, balanced, and versatile, with enough range for complex product systems, expressive visuals, and layered brand moments.

## Color Palette

- **Midnight Magenta**: `#0B090A` - Foundation color for headers, navigation, and high-contrast surfaces
- **Midnight Sky Blue**: `#161A1D` - Primary interface accents, links, charts, and trust-building UI elements
- **Deep Rose**: `#660708` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Rich Rose**: `#A4161A` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Vivid Rose**: `#BA181B` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Vivid Rose 2**: `#E5383B` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Warm Red**: `#B1A7A6` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Silver Gray**: `#D3D3D3` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Soft White**: `#F5F3F4` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Crisp White**: `#FFFFFF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones

## Typography

- **Headers**: Inter Display
- **Body Text**: Public Sans
- **Font Direction**: Inter Display and Public Sans support a serious enterprise feel with authoritative, dependable readability.

## Best Used For

Enterprise dashboards, design systems, compliance software, architecture portfolios, data products, and documentation.
