# Summer Ocean Breeze

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Vivid Rose**: `#E63946` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Pale Green**: `#F1FAEE` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Pale Teal**: `#A8DADC` - Primary interface accents, links, charts, and trust-building UI elements
- **Balanced Cyan**: `#457B9D` - Primary interface accents, links, charts, and trust-building UI elements
- **Deep Sky Blue**: `#1D3557` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
