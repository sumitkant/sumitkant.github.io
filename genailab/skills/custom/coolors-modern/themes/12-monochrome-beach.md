# Monochrome Beach

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Charcoal**: `#353535` - Foundation color for headers, navigation, and high-contrast surfaces
- **Muted Teal**: `#3C6E71` - Primary interface accents, links, charts, and trust-building UI elements
- **Crisp White**: `#FFFFFF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Soft White**: `#D9D9D9` - Soft backgrounds, highlight panels, and secondary content blocks
- **Muted Cyan**: `#284B63` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Inter Display
- **Body Text**: Public Sans
- **Font Direction**: Inter Display and Public Sans support a serious enterprise feel with authoritative, dependable readability.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
