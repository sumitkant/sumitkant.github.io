# Vintage Vibes

A color system that feels soft, nostalgic, and editorial, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Pale Orange**: `#F1DAC4` - Soft backgrounds, highlight panels, and secondary content blocks
- **Soft Violet**: `#A69CAC` - Creative accents, hero graphics, product moments, and expressive highlights
- **Muted Blue**: `#474973` - Primary interface accents, links, charts, and trust-building UI elements
- **Midnight Blue**: `#161B33` - Primary interface accents, links, charts, and trust-building UI elements
- **Midnight Blue 2**: `#0D0C1D` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: DM Sans
- **Body Text**: Source Sans 3
- **Font Direction**: DM Sans brings warmth and humanity to the earthy palette, while Source Sans 3 supports mature, editorial body content.

## Best Used For

Editorial brands, boutique hospitality, storytelling decks, maker products, calm landing pages, and content-rich websites.
