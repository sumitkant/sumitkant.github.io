# Monochrome Red

A color system that feels structured, mature, and product-focused, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Deep Blue**: `#2B2D42` - Foundation color for headers, navigation, and high-contrast surfaces
- **Cool Sky Blue**: `#8D99AE` - Primary interface accents, links, charts, and trust-building UI elements
- **Soft White**: `#EDF2F4` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Vivid Rose**: `#EF233C` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Vivid Rose 2**: `#D80032` - Alert accents, energetic CTAs, campaign details, and emotional highlights

## Typography

- **Headers**: Inter Display
- **Body Text**: Public Sans
- **Font Direction**: Inter Display and Public Sans support a serious enterprise feel with authoritative, dependable readability.

## Best Used For

Enterprise dashboards, design systems, compliance software, architecture portfolios, data products, and documentation.
