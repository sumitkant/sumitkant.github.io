# Industrial Blue

A color system that feels structured, mature, and product-focused, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Soft White**: `#DCDCDD` - Soft backgrounds, highlight panels, and secondary content blocks
- **Silver Gray**: `#C5C3C6` - Creative accents, hero graphics, product moments, and expressive highlights
- **Charcoal**: `#46494C` - Primary interface accents, links, charts, and trust-building UI elements
- **Muted Sky Blue**: `#4C5C68` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Cyan**: `#1985A1` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Inter Display
- **Body Text**: Public Sans
- **Font Direction**: Inter Display and Public Sans support a serious enterprise feel with authoritative, dependable readability.

## Best Used For

Enterprise dashboards, design systems, compliance software, architecture portfolios, data products, and documentation.
