# Deep Sea

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Midnight Sky Blue**: `#13293D` - Foundation color for headers, navigation, and high-contrast surfaces
- **Rich Cyan**: `#006494` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Cyan 2**: `#247BA0` - Primary interface accents, links, charts, and trust-building UI elements
- **Vivid Cyan**: `#1B98E0` - Primary interface accents, links, charts, and trust-building UI elements
- **Pale Teal**: `#E8F1F2` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
