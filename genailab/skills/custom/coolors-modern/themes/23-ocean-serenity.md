# Ocean Serenity

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Midnight Cyan**: `#01161E` - Foundation color for headers, navigation, and high-contrast surfaces
- **Deep Cyan**: `#124559` - Primary interface accents, links, charts, and trust-building UI elements
- **Balanced Cyan**: `#598392` - Primary interface accents, links, charts, and trust-building UI elements
- **Soft Olive**: `#AEC3B0` - Success states, nature cues, wellness accents, and calm supporting elements
- **Pale Lime**: `#EFF6E0` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
