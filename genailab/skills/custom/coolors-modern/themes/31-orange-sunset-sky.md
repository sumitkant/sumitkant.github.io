# Orange Sunset Sky

A color system that feels warm, cinematic, and optimistic, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Deep Blue**: `#2D3142` - Foundation color for headers, navigation, and high-contrast surfaces
- **Muted Sky Blue**: `#4F5D75` - Primary interface accents, links, charts, and trust-building UI elements
- **Silver Gray**: `#BFC0C0` - Primary interface accents, links, charts, and trust-building UI elements
- **Crisp White**: `#FFFFFF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Warm Coral**: `#EF8354` - Supporting accent for illustrations, dividers, icon fills, and content hierarchy

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans offers a premium, modern, and approachable headline voice, while Inter keeps the body system highly readable.

## Best Used For

Lifestyle campaigns, media kits, event branding, restaurant concepts, travel stories, and warm marketing presentations.
