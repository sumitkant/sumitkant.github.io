# Ocean Breeze

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Muted Sky Blue**: `#495867` - Foundation color for headers, navigation, and high-contrast surfaces
- **Balanced Sky Blue**: `#577399` - Primary interface accents, links, charts, and trust-building UI elements
- **Pale Sky Blue**: `#BDD5EA` - Primary interface accents, links, charts, and trust-building UI elements
- **Pale Blue**: `#F7F7FF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Warm Red**: `#FE5F55` - Alert accents, energetic CTAs, campaign details, and emotional highlights

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
