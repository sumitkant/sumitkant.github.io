# Magenta Dream

A broad-spectrum color system that feels expressive, creative, and high-energy, with enough range for complex product systems, expressive visuals, and layered brand moments.

## Color Palette

- **Rich Magenta**: `#B7094C` - Foundation color for headers, navigation, and high-contrast surfaces
- **Rich Magenta 2**: `#A01A58` - Deep contrast layer for footers, app shells, and premium dark sections
- **Rich Magenta 3**: `#892B64` - Creative accents, hero graphics, product moments, and expressive highlights
- **Muted Purple**: `#723C70` - Creative accents, hero graphics, product moments, and expressive highlights
- **Muted Indigo**: `#5C4D7D` - Deep contrast layer for footers, app shells, and premium dark sections
- **Balanced Sky Blue**: `#455E89` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Cyan**: `#2E6F95` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Cyan 2**: `#1780A1` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Teal**: `#0091AD` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Space Grotesk
- **Body Text**: Inter
- **Font Direction**: Space Grotesk gives the saturated violet and blue range an innovative, AI-native edge, while Inter keeps dense UI copy efficient and trustworthy.

## Best Used For

GenAI tools, creative platforms, launch pages, entertainment brands, social campaigns, and expressive dashboards.
