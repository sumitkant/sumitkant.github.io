# Bold Hues

A color system that feels expressive, creative, and high-energy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Vivid Magenta**: `#F72585` - Supporting accent for illustrations, dividers, icon fills, and content hierarchy
- **Rich Violet**: `#7209B7` - Creative accents, hero graphics, product moments, and expressive highlights
- **Rich Indigo**: `#3A0CA3` - Deep contrast layer for footers, app shells, and premium dark sections
- **Cool Blue**: `#4361EE` - Primary interface accents, links, charts, and trust-building UI elements
- **Cool Cyan**: `#4CC9F0` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Poppins
- **Body Text**: Figtree
- **Font Direction**: Poppins matches the energetic, marketing-forward color contrast, while Figtree keeps body copy casual, modern, and approachable.

## Best Used For

GenAI tools, creative platforms, launch pages, entertainment brands, social campaigns, and expressive dashboards.
