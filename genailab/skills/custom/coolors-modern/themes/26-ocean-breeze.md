# Ocean Breeze

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Rich Cyan**: `#16697A` - Foundation color for headers, navigation, and high-contrast surfaces
- **Balanced Cyan**: `#489FB5` - Primary interface accents, links, charts, and trust-building UI elements
- **Cool Teal**: `#82C0CC` - Primary interface accents, links, charts, and trust-building UI elements
- **Pale Coral**: `#EDE7E3` - Soft backgrounds, highlight panels, and secondary content blocks
- **Warm Orange**: `#FFA62B` - Supporting accent for illustrations, dividers, icon fills, and content hierarchy

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
