# Purple Raindrops

A broad-spectrum color system that feels expressive, creative, and high-energy, with enough range for complex product systems, expressive visuals, and layered brand moments.

- **Vivid Magenta**: `#F72585` - Supporting accent for illustrations, dividers, icon fills, and content hierarchy
- **Vivid Purple**: `#B5179E` - Creative accents, hero graphics, product moments, and expressive highlights
- **Rich Violet**: `#7209B7` - Creative accents, hero graphics, product moments, and expressive highlights
- **Rich Indigo**: `#560BAD` - Creative accents, hero graphics, product moments, and expressive highlights
- **Rich Indigo 2**: `#480CA8` - Creative accents, hero graphics, product moments, and expressive highlights
- **Rich Indigo 3**: `#3A0CA3` - Deep contrast layer for footers, app shells, and premium dark sections
- **Balanced Blue**: `#3F37C9` - Primary interface accents, links, charts, and trust-building UI elements
- **Cool Blue**: `#4361EE` - Primary interface accents, links, charts, and trust-building UI elements
- **Cool Sky Blue**: `#4895EF` - Primary interface accents, links, charts, and trust-building UI elements
- **Cool Cyan**: `#4CC9F0` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Space Grotesk
- **Body Text**: Inter
- **Font Direction**: Space Grotesk gives the saturated violet and blue range an innovative, AI-native edge, while Inter keeps dense UI copy efficient and trustworthy.

## Best Used For

GenAI tools, creative platforms, launch pages, entertainment brands, social campaigns, and expressive dashboards.
