# Deep Sea

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Crisp White**: `#FFFFFF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Midnight Cyan**: `#00171F` - Primary interface accents, links, charts, and trust-building UI elements
- **Deep Cyan**: `#003459` - Primary interface accents, links, charts, and trust-building UI elements
- **Rich Cyan**: `#007EA7` - Primary interface accents, links, charts, and trust-building UI elements
- **Vivid Cyan**: `#00A8E8` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Plus Jakarta Sans
- **Body Text**: Inter
- **Font Direction**: Plus Jakarta Sans adds premium modernity to the cool aquatic palette, while Inter provides a trustworthy product-reading experience.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
