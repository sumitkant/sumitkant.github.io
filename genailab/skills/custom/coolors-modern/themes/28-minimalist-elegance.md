# Minimalist Elegance

A color system that feels polished, premium, and refined, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Deep Blue**: `#2D3142` - Foundation color for headers, navigation, and high-contrast surfaces
- **Silver Gray**: `#BFC0C0` - Primary interface accents, links, charts, and trust-building UI elements
- **Crisp White**: `#FFFFFF` - Clean page backgrounds, cards, spacious UI areas, and reversed text zones
- **Warm Coral**: `#EF8354` - Supporting accent for illustrations, dividers, icon fills, and content hierarchy
- **Muted Sky Blue**: `#4F5D75` - Primary interface accents, links, charts, and trust-building UI elements

## Typography

- **Headers**: Inter Display
- **Body Text**: Public Sans
- **Font Direction**: Inter Display and Public Sans support a serious enterprise feel with authoritative, dependable readability.

## Best Used For

Premium SaaS, fintech, luxury packaging, executive presentations, portfolio sites, and polished product launches.
