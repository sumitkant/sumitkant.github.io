# Azure Skies

A color system that feels cool, coastal, and trustworthy, balancing strong brand accents with usable neutrals for polished digital interfaces and presentation design.

## Color Palette

- **Midnight Purple**: `#080708` - Foundation color for headers, navigation, and high-contrast surfaces
- **Cool Sky Blue**: `#3772FF` - Primary interface accents, links, charts, and trust-building UI elements
- **Vivid Rose**: `#DF2935` - Alert accents, energetic CTAs, campaign details, and emotional highlights
- **Warm Amber**: `#FDCA40` - Warm calls-to-action, badges, pricing highlights, and emphasis states
- **Soft White**: `#E6E8E6` - Soft backgrounds, highlight panels, and secondary content blocks

## Typography

- **Headers**: Poppins
- **Body Text**: Figtree
- **Font Direction**: Poppins matches the energetic, marketing-forward color contrast, while Figtree keeps body copy casual, modern, and approachable.

## Best Used For

Travel products, wellness apps, hospitality branding, clean SaaS dashboards, and coastal lifestyle campaigns.
