# Autumn Orchard

A organic and restorative theme built from orange and green tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Bright Orange**: `#DAA520` - Chosen page and canvas background
- **Body Text / Deep Pink**: `#660033` - Primary readable text color
- **Primary Accent / Dark Green**: `#2E6F40` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Orange**: `#BEA58E` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
