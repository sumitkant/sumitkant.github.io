# Blue Eclipse

A airy and tranquil theme built from blue tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Muted Blue**: `#8686AC` - Chosen page and canvas background
- **Body Text / Deep Blue**: `#0F0E47` - Primary readable text color
- **Primary Accent / Deep Blue**: `#272757` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Blue**: `#505081` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
