# Fireworks

A bold and high-impact theme built from red and orange-red tones that conveys experimentation, momentum, and attention.

## Color Palette

- **Background / Dark Red**: `#960016` - Chosen page and canvas background
- **Body Text / Bright Coral**: `#E88A38` - Primary readable text color
- **Primary Accent / Bright Red**: `#ED2100` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Magenta**: `#BF00AC` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Space Grotesk Bold
- **Body Text**: Inter

## Best Used For

AI products, gaming interfaces, music apps, creative campaigns, high-energy landing pages.
