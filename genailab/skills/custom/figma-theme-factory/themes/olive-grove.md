# Olive Grove

A organic and restorative theme built from gold, green, and yellow tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Muted Coral**: `#F2F0EF` - Chosen page and canvas background
- **Body Text / Dark Yellow**: `#636B2F` - Primary readable text color
- **Primary Accent / Dark Gold**: `#808000` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Green**: `#228B22` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
