# Sharp Edge

A bold and high-impact theme built from gray, light gray, and red tones that conveys experimentation, momentum, and attention.

## Color Palette

- **Background / Bright Mint**: `#4DFFBC` - Chosen page and canvas background
- **Body Text / Gray**: `#898989` - Primary readable text color
- **Primary Accent / Bright Red**: `#FF4D4D` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Gray**: `#D9D9D9` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Space Grotesk Bold
- **Body Text**: Inter

## Best Used For

Entertainment platforms, creator tools, youth brands, launch campaigns, experimental web experiences.
