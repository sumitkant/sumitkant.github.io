# Lush Forest

A organic and restorative theme built from green tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Pale Green**: `#CFFFDC` - Chosen page and canvas background
- **Body Text / Muted Green**: `#253D2C` - Primary readable text color
- **Primary Accent / Dark Green**: `#2E6F40` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Green**: `#68BA7F` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
