# Cactus Flower

A soft and expressive theme built from pink and green tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Green**: `#92E4BA` - Chosen page and canvas background
- **Body Text / Muted Pink**: `#845763` - Primary readable text color
- **Primary Accent / Light Pink**: `#E491A6` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Green**: `#90C67F` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
