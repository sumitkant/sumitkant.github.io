# Wildflower Meadow

A organic and restorative theme built from gold, orange, and cyan tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Pale Gold**: `#FBF9E7` - Chosen page and canvas background
- **Body Text / Light Cyan**: `#82C8E5` - Primary readable text color
- **Primary Accent / Bright Lime**: `#7CFC00` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Orange**: `#FDB813` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
