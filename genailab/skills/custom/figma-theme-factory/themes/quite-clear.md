# Quite Clear

A balanced and refined theme built from light gray, mint, and red tones that conveys restraint, trust, and quiet sophistication.

## Color Palette

- **Background / White**: `#F2F2F2` - Chosen page and canvas background
- **Body Text / Deep Red**: `#4D1717` - Primary readable text color
- **Primary Accent / Deep Mint**: `#174D38` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Gray**: `#CBCBCB` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Inter Display Bold
- **Body Text**: Public Sans

## Best Used For

Corporate presentations, editorial layouts, premium retail, consulting reports, minimalist product interfaces.
