# Cotton Candy Skies

A soft and expressive theme built from indigo, teal, and pink tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Teal**: `#B8E3E9` - Chosen page and canvas background
- **Body Text / Light Indigo**: `#B298E7` - Primary readable text color
- **Primary Accent / Pale Pink**: `#F9BEDD` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Pink**: `#F5B8D5` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
