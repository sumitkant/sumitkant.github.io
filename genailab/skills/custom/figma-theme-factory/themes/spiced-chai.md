# Spiced Chai

A soft and expressive theme built from gold and orange tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Gold**: `#FDFBD4` - Chosen page and canvas background
- **Body Text / Dark Orange**: `#825E34` - Primary readable text color
- **Primary Accent / Soft Orange**: `#D47E30` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Orange**: `#8D5A2B` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
