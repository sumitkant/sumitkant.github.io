# Iris Garden

A soft and expressive theme built from violet and green tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Muted Green**: `#82AB7D` - Chosen page and canvas background
- **Body Text / Dark Violet**: `#692475` - Primary readable text color
- **Primary Accent / Muted Violet**: `#A47DAB` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Yellow**: `#6A6B4E` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
