# Blooming Romance

A soft and expressive theme built from pink and lime tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Soft Pink**: `#E673AC` - Chosen page and canvas background
- **Body Text / Deep Pink**: `#660033` - Primary readable text color
- **Primary Accent / Deep Green**: `#00520A` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Lime**: `#469110` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
