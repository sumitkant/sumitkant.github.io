# Wisteria Bloom

A soft and expressive theme built from blue, violet, and magenta tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Blue**: `#D3D3FF` - Chosen page and canvas background
- **Body Text / Bright Violet**: `#9400D3` - Primary readable text color
- **Primary Accent / Light Magenta**: `#ED80E9` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Magenta**: `#D8BFD8` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
