# Space Berries

A bold and high-impact theme built from magenta, pink, and orange-red tones that conveys experimentation, momentum, and attention.

## Color Palette

- **Background / Dark Pink**: `#8C1946` - Chosen page and canvas background
- **Body Text / Pale Pink**: `#FFB8DC` - Primary readable text color
- **Primary Accent / Bright Magenta**: `#FD3DB5` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Coral**: `#FB6A2C` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Space Grotesk Bold
- **Body Text**: Inter

## Best Used For

AI products, gaming interfaces, music apps, creative campaigns, high-energy landing pages.
