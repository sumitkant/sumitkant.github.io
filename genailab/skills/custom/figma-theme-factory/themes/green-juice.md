# Green Juice

A organic and restorative theme built from lime tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Bright Lime**: `#4CBB17` - Chosen page and canvas background
- **Body Text / Muted Lime**: `#293325` - Primary readable text color
- **Primary Accent / Dark Lime**: `#48872B` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Lime**: `#39542C` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
