# Frozen Lake

A airy and tranquil theme built from cyan, teal, and red tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Pale Red**: `#FFFAFA` - Chosen page and canvas background
- **Body Text / Dark Blue**: `#000080` - Primary readable text color
- **Primary Accent / Light Teal**: `#ADD8E6` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Cyan**: `#6D8196` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
