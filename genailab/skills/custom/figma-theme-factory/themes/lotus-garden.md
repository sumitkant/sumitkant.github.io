# Lotus Garden

A soft and expressive theme built from green, red, and magenta tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Green**: `#ADEBB3` - Chosen page and canvas background
- **Body Text / Dark Red**: `#6B403C` - Primary readable text color
- **Primary Accent / Light Red**: `#FF857A` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Magenta**: `#EBAEE6` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
