# Morning Dew

A airy and tranquil theme built from teal, green, and light gray tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Light Teal**: `#AFEEEE` - Chosen page and canvas background
- **Body Text / Light Gray**: `#C4C4C4` - Primary readable text color
- **Primary Accent / Light Green**: `#ADEBB3` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Gray**: `#D3D3D3` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
