# Spring Energy

A organic and restorative theme built from lime, gold, and mint tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Bright Gold**: `#FFFC30` - Chosen page and canvas background
- **Body Text / Dark Green**: `#00BF33` - Primary readable text color
- **Primary Accent / Bright Lime**: `#89F336` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Mint**: `#4FE0CB` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
