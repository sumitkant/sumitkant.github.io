# Hydrangea

A soft and expressive theme built from pink, red, and magenta tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Red**: `#FFC2BA` - Chosen page and canvas background
- **Body Text / Soft Violet**: `#AD56C4` - Primary readable text color
- **Primary Accent / Light Pink**: `#FF8DA1` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Magenta**: `#FF9CE9` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
