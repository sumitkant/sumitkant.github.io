# Minty Fresh

A airy and tranquil theme built from mint tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Pale Mint**: `#BFFFED` - Chosen page and canvas background
- **Body Text / Muted Mint**: `#558B71` - Primary readable text color
- **Primary Accent / Light Mint**: `#98FBCB` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Mint**: `#7FCFA8` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
