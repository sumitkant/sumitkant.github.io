# Tuscan Sunset

A soft and expressive theme built from red, orange, and violet tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Orange**: `#FFD3AC` - Chosen page and canvas background
- **Body Text / Dark Coral**: `#8A2B0E` - Primary readable text color
- **Primary Accent / Bright Red**: `#E35336` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Violet**: `#9988A1` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
