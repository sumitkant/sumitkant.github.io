# Country Garden

A soft and expressive theme built from gold and indigo tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Gold**: `#FFFFE3` - Chosen page and canvas background
- **Body Text / Dark Violet**: `#723480` - Primary readable text color
- **Primary Accent / Pale Indigo**: `#DBD4FF` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Gold**: `#808034` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
