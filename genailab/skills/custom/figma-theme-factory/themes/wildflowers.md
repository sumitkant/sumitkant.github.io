# Wildflowers

A soft and expressive theme built from green and red tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Green**: `#A8DCAB` - Chosen page and canvas background
- **Body Text / Soft Green**: `#519755` - Primary readable text color
- **Primary Accent / Light Red**: `#DBAAA7` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Magenta**: `#BE91BE` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
