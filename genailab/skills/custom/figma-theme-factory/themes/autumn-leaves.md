# Autumn Leaves

A organic and restorative theme built from orange tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Deep Orange**: `#614419` - Chosen page and canvas background
- **Body Text / Bright Orange**: `#FFB343` - Primary readable text color
- **Primary Accent / Soft Orange**: `#DB9A39` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Orange**: `#B37E2E` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
