# Golden Hour

A soft and expressive theme built from orange, gold, and violet tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Gold**: `#ECE9C8` - Chosen page and canvas background
- **Body Text / Soft Gold**: `#CFB53B` - Primary readable text color
- **Primary Accent / Light Violet**: `#E0B0FF` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Orange**: `#FFBF00` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
