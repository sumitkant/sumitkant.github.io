# Retro Calm

A airy and tranquil theme built from mint, orange-red, and gold tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Soft Gold**: `#D7D982` - Chosen page and canvas background
- **Body Text / Soft Indigo**: `#AE82D9` - Primary readable text color
- **Primary Accent / Soft Coral**: `#D99E82` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Mint**: `#81D8D0` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
