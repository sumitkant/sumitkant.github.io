# Winter Berry

A organic and restorative theme built from red, green, and teal tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Dark Red**: `#8E0A1E` - Chosen page and canvas background
- **Body Text / White**: `#FFFFFF` - Primary readable text color
- **Primary Accent / Light Teal**: `#ADD8E6` - Buttons, links, and key visual emphasis
- **Secondary Accent / Dark Green**: `#2E6F40` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
