# Golden Taupe

A polished and reliable theme built from gold tones that conveys clarity, confidence, and professional calm.

## Color Palette

- **Background / Pale Gold**: `#FDFBD4` - Chosen page and canvas background
- **Body Text / Soft Orange**: `#CE8946` - Primary readable text color
- **Primary Accent / Soft Gold**: `#D4AF37` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Gold**: `#BDB76B` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Plus Jakarta Sans Bold
- **Body Text**: Inter

## Best Used For

SaaS dashboards, professional reports, knowledge products, fintech tools, enterprise websites.
