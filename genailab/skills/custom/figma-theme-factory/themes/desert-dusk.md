# Desert Dusk

A soft and expressive theme built from red, orange-red, and pink tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Bright Coral**: `#E68057` - Chosen page and canvas background
- **Body Text / Soft Magenta**: `#993A8B` - Primary readable text color
- **Primary Accent / Soft Pink**: `#BF7587` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Red**: `#A2574F` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
