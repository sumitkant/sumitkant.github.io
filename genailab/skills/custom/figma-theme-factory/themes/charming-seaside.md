# Charming Seaside

A airy and tranquil theme built from teal and green tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Pale Mint**: `#C0FDF2` - Chosen page and canvas background
- **Body Text / Soft Teal**: `#85D1DB` - Primary readable text color
- **Primary Accent / Light Teal**: `#B3EBF2` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Green**: `#B6F2D1` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
