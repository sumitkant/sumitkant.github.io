# Summer Breeze

A organic and restorative theme built from gold, red, and cyan tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Bright Gold**: `#FFEB3B` - Chosen page and canvas background
- **Body Text / Light Red**: `#F88379` - Primary readable text color
- **Primary Accent / Light Cyan**: `#82C8E5` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Orange**: `#E6D8C4` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
