# Evening Rose

A soft and expressive theme built from red and magenta tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Red**: `#DCA1A1` - Chosen page and canvas background
- **Body Text / Charcoal**: `#4A4A4A` - Primary readable text color
- **Primary Accent / Soft Magenta**: `#8E4585` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Red**: `#996666` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
