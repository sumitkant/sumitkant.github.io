# Siltstone

A balanced and refined theme built from gold and orange-red tones that conveys restraint, trust, and quiet sophistication.

## Color Palette

- **Background / Pale Gold**: `#FFF5B8` - Chosen page and canvas background
- **Body Text / Soft Orange**: `#CCA25A` - Primary readable text color
- **Primary Accent / Light Coral**: `#FFB16E` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Gold**: `#CBBD93` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Inter Display Bold
- **Body Text**: Public Sans

## Best Used For

Corporate presentations, editorial layouts, premium retail, consulting reports, minimalist product interfaces.
