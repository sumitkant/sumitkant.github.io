# Stone Path

A balanced and refined theme built from orange and gold tones that conveys restraint, trust, and quiet sophistication.

## Color Palette

- **Background / Muted Orange**: `#E8E5DF` - Chosen page and canvas background
- **Body Text / Muted Orange**: `#968F83` - Primary readable text color
- **Primary Accent / Muted Orange**: `#A49A87` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Gold**: `#A5A58D` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Inter Display Bold
- **Body Text**: Public Sans

## Best Used For

Corporate presentations, editorial layouts, premium retail, consulting reports, minimalist product interfaces.
