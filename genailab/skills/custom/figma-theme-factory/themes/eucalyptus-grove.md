# Eucalyptus Grove

A organic and restorative theme built from gold, gray, and orange-red tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Muted Coral**: `#F2F0EF` - Chosen page and canvas background
- **Body Text / Muted Green**: `#4B6E48` - Primary readable text color
- **Primary Accent / Muted Gold**: `#B2AC88` - Buttons, links, and key visual emphasis
- **Secondary Accent / Gray**: `#898989` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
