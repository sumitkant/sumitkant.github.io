# Mossy Hollow

A organic and restorative theme built from yellow tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Light Yellow**: `#D4DE95` - Chosen page and canvas background
- **Body Text / Muted Yellow**: `#3D4127` - Primary readable text color
- **Primary Accent / Dark Yellow**: `#636B2F` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Yellow**: `#BAC095` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
