# Honeycomb

A balanced and refined theme built from gold tones that conveys restraint, trust, and quiet sophistication.

## Color Palette

- **Background / Pale Gold**: `#FFFDD0` - Chosen page and canvas background
- **Body Text / Dark Coral**: `#895129` - Primary readable text color
- **Primary Accent / Bright Gold**: `#FFC107` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Gold**: `#F9E076` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Inter Display Bold
- **Body Text**: Public Sans

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
