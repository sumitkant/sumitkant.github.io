# Winter Chill

A organic and restorative theme built from teal tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Light Teal**: `#B8E3E9` - Chosen page and canvas background
- **Body Text / Deep Teal**: `#0B2E33` - Primary readable text color
- **Primary Accent / Muted Teal**: `#4F7C82` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Teal**: `#93B1B5` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
