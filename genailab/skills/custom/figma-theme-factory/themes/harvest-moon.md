# Harvest Moon

A organic and restorative theme built from pink, gold, and red tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Dark Pink**: `#800020` - Chosen page and canvas background
- **Body Text / Bright Gold**: `#FFCE1B` - Primary readable text color
- **Primary Accent / Dark Coral**: `#BE5103` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Red**: `#A52A2A` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
