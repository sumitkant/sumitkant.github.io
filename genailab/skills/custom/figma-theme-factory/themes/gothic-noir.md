# Gothic Noir

A bold and high-impact theme built from light gray and red tones that conveys experimentation, momentum, and attention.

## Color Palette

- **Background / Black**: `#000000` - Chosen page and canvas background
- **Body Text / Light Gray**: `#D1D0D0` - Primary readable text color
- **Primary Accent / Muted Red**: `#5C4E4E` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Red**: `#988686` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Space Grotesk Bold
- **Body Text**: Inter

## Best Used For

AI products, gaming interfaces, music apps, creative campaigns, high-energy landing pages.
