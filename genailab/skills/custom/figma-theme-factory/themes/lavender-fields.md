# Lavender Fields

A soft and expressive theme built from gold and blue tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Pale Gold**: `#FDFBD4` - Chosen page and canvas background
- **Body Text / Light Violet**: `#CF6DFC` - Primary readable text color
- **Primary Accent / Pale Blue**: `#C1BFFF` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Gold**: `#BDB96A` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
