# Burnt Sienna

A polished and reliable theme built from red, gold, and orange-red tones that conveys clarity, confidence, and professional calm.

## Color Palette

- **Background / Pale Gold**: `#F5F5DC` - Chosen page and canvas background
- **Body Text / Soft Coral**: `#A0522D` - Primary readable text color
- **Primary Accent / Bright Coral**: `#F4A460` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Red**: `#E35336` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Plus Jakarta Sans Bold
- **Body Text**: Inter

## Best Used For

SaaS dashboards, professional reports, knowledge products, fintech tools, enterprise websites.
