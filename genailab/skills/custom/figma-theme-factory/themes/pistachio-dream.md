# Pistachio Dream

A bright and energetic theme built from green, yellow, and lime tones that conveys optimism, motion, and approachability.

## Color Palette

- **Background / Light Yellow**: `#E3F0A3` - Chosen page and canvas background
- **Body Text / Soft Green**: `#42D674` - Primary readable text color
- **Primary Accent / Light Green**: `#80EF80` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Lime**: `#BADBA2` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Poppins Bold
- **Body Text**: Figtree

## Best Used For

Consumer apps, children's products, edtech, travel promotions, social campaigns, lively dashboards.
