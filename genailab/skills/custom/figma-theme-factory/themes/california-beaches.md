# California Beaches

A bright and energetic theme built from orange, teal, and cyan tones that conveys optimism, motion, and approachability.

## Color Palette

- **Background / Bright Teal**: `#66F4FF` - Chosen page and canvas background
- **Body Text / Muted Cyan**: `#7D99AA` - Primary readable text color
- **Primary Accent / Bright Cyan**: `#66C4FF` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Orange**: `#FFC067` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Poppins Bold
- **Body Text**: Figtree

## Best Used For

Consumer apps, children's products, edtech, travel promotions, social campaigns, lively dashboards.
