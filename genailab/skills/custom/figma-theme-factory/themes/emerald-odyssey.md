# Emerald Odyssey

A airy and tranquil theme built from mint tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Soft Mint**: `#73E6CB` - Chosen page and canvas background
- **Body Text / Deep Mint**: `#0A3C30` - Primary readable text color
- **Primary Accent / Deep Mint**: `#00674F` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Mint**: `#3EBB9E` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
