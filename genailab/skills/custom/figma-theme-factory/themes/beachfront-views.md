# Beachfront Views

A airy and tranquil theme built from gold and indigo tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Pale Gold**: `#EDE8D0` - Chosen page and canvas background
- **Body Text / Dark Gold**: `#6E632E` - Primary readable text color
- **Primary Accent / Light Blue**: `#ABBEED` - Buttons, links, and key visual emphasis
- **Secondary Accent / Pale Indigo**: `#DBD1ED` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
