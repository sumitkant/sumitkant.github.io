# Cappuccino

A balanced and refined theme built from orange tones that conveys restraint, trust, and quiet sophistication.

## Color Palette

- **Background / Muted Orange**: `#C6C0B9` - Chosen page and canvas background
- **Body Text / Deep Orange**: `#422701` - Primary readable text color
- **Primary Accent / Soft Orange**: `#D6B588` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Orange**: `#705E46` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Plus Jakarta Sans Bold
- **Body Text**: Inter

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
