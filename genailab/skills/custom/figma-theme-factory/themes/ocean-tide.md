# Ocean Tide

A airy and tranquil theme built from teal and blue tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Bright Teal**: `#00D1D1` - Chosen page and canvas background
- **Body Text / Deep Teal**: `#005C5C` - Primary readable text color
- **Primary Accent / Dark Teal**: `#00ADAD` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Blue**: `#4052D6` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
