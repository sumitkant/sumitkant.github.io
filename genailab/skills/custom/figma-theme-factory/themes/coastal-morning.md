# Coastal Morning

A airy and tranquil theme built from lime, cyan, and gold tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Pale Gold**: `#F5F5DC` - Chosen page and canvas background
- **Body Text / Soft Lime**: `#93C572` - Primary readable text color
- **Primary Accent / Light Cyan**: `#89CFF0` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Cyan**: `#82C8E5` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
