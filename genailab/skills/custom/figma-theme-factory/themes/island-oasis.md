# Island Oasis

A organic and restorative theme built from yellow, gold, and lime tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Light Gold**: `#FFF37E` - Chosen page and canvas background
- **Body Text / Bright Orange**: `#FFAE25` - Primary readable text color
- **Primary Accent / Bright Yellow**: `#CCFF00` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Lime**: `#9DE05A` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
