# Fresh Peach

A soft and expressive theme built from orange, red, and orange-red tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Orange**: `#FFD3AC` - Chosen page and canvas background
- **Body Text / Soft Coral**: `#E39A7B` - Primary readable text color
- **Primary Accent / Light Red**: `#FFB5AB` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Orange**: `#DBB06B` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
