# Tropical Rainforest

A organic and restorative theme built from green and yellow tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Soft Green**: `#50C878` - Chosen page and canvas background
- **Body Text / Dark Green**: `#4F7942` - Primary readable text color
- **Primary Accent / Soft Lime**: `#71AA34` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Yellow**: `#7E8C54` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
