# Lavender Lullaby

A soft and expressive theme built from blue and teal tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Teal**: `#9EF0FF` - Chosen page and canvas background
- **Body Text / Soft Indigo**: `#8E70CF` - Primary readable text color
- **Primary Accent / Light Blue**: `#A4A5F5` - Buttons, links, and key visual emphasis
- **Secondary Accent / Light Blue**: `#B5C7EB` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
