# Calm Blue

A airy and tranquil theme built from cyan tones that conveys ease, focus, and gentle momentum.

## Color Palette

- **Background / Light Cyan**: `#90D5FF` - Chosen page and canvas background
- **Body Text / Soft Cyan**: `#517891` - Primary readable text color
- **Primary Accent / Bright Cyan**: `#57B0FF` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Cyan**: `#77B1D4` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Lexend Bold
- **Body Text**: Inter

## Best Used For

Wellness apps, travel guides, healthcare content, productivity tools, clean editorial experiences.
