# Jewel Box

A bold and high-impact theme built from green, blue, and indigo tones that conveys experimentation, momentum, and attention.

## Color Palette

- **Background / Dark Blue**: `#0F52BA` - Chosen page and canvas background
- **Body Text / Soft Gold**: `#CFB53B` - Primary readable text color
- **Primary Accent / Soft Green**: `#50C878` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Indigo**: `#9966CC` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Space Grotesk Bold
- **Body Text**: Inter

## Best Used For

Entertainment platforms, creator tools, youth brands, launch campaigns, experimental web experiences.
