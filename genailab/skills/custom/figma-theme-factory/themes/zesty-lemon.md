# Zesty Lemon

A bright and energetic theme built from gold tones that conveys optimism, motion, and approachability.

## Color Palette

- **Background / Bright Gold**: `#FFFF66` - Chosen page and canvas background
- **Body Text / Soft Gold**: `#B3B347` - Primary readable text color
- **Primary Accent / Bright Gold**: `#FFE566` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Gold**: `#D6D58B` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Poppins Bold
- **Body Text**: Figtree

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
