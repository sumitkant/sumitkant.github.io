# Stormy Morning

A polished and reliable theme built from cyan tones that conveys clarity, confidence, and professional calm.

## Color Palette

- **Background / Pale Cyan**: `#BDDDFC` - Chosen page and canvas background
- **Body Text / Muted Cyan**: `#384959` - Primary readable text color
- **Primary Accent / Light Cyan**: `#88BDF2` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Cyan**: `#6A89A7` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Plus Jakarta Sans Bold
- **Body Text**: Inter

## Best Used For

SaaS dashboards, professional reports, knowledge products, fintech tools, enterprise websites.
