# Soft Spring

A organic and restorative theme built from blue, cyan, and green tones that conveys freshness, connection, and grounded energy.

## Color Palette

- **Background / Soft Mint**: `#85DECB` - Chosen page and canvas background
- **Body Text / Bright Blue**: `#6395EE` - Primary readable text color
- **Primary Accent / Light Cyan**: `#90B8D6` - Buttons, links, and key visual emphasis
- **Secondary Accent / Soft Green**: `#88CFA8` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Sustainability reports, outdoor brands, wellness products, agriculture, ecological campaigns, nature-focused portfolios.
