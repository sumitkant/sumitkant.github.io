# Cherry Blossom

A soft and expressive theme built from red, green, and pink tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / White**: `#FFFFFF` - Chosen page and canvas background
- **Body Text / Pale Pink**: `#FFB7C5` - Primary readable text color
- **Primary Accent / Pale Red**: `#F2C7C7` - Buttons, links, and key visual emphasis
- **Secondary Accent / Pale Green**: `#D5F3D8` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
