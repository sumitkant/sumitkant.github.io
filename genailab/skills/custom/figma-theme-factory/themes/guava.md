# Guava

A bright and energetic theme built from orange-red and red tones that conveys optimism, motion, and approachability.

## Color Palette

- **Background / Light Coral**: `#FFB578` - Chosen page and canvas background
- **Body Text / Soft Pink**: `#CF5376` - Primary readable text color
- **Primary Accent / Bright Coral**: `#FF8559` - Buttons, links, and key visual emphasis
- **Secondary Accent / Bright Red**: `#E65447` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: Poppins Bold
- **Body Text**: Figtree

## Best Used For

Food and beverage brands, hospitality decks, lifestyle campaigns, restaurant websites, seasonal promotions.
