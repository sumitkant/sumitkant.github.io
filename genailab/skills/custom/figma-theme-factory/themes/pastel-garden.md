# Pastel Garden

A soft and expressive theme built from pink, red, and lime tones that conveys warmth, elegance, and invitation.

## Color Palette

- **Background / Light Red**: `#F0B8B8` - Chosen page and canvas background
- **Body Text / Muted Coral**: `#54463A` - Primary readable text color
- **Primary Accent / Soft Pink**: `#C75F71` - Buttons, links, and key visual emphasis
- **Secondary Accent / Muted Lime**: `#A2AE9D` - Highlights, badges, charts, and supporting details

## Typography

- **Headers**: DM Sans Bold
- **Body Text**: Source Sans 3

## Best Used For

Beauty brands, event planning, wedding websites, fashion lookbooks, lifestyle and wellness content.
